## 🎨 原型页面导航

### 核心页面
1. **[登录页](../prototypes/login.html)** - 系统登录入口
2. **[首页/仪表盘](../prototypes/dashboard.html)** - 数据概览和快速操作
3. **[爬取任务列表](../prototypes/crawl-tasks.html)** - 查看和管理所有爬取任务
4. **[创建爬取任务](../prototypes/create_task.html)** - 创建新的爬取任务
5. **[任务详情](../prototypes/task_detail.html)** - 查看任务详细信息和执行日志
6. **[数据查询](../prototypes/data_query.html)** - 查询和筛选酒店数据
7. **[数据详情](../prototypes/data_detail.html)** - 查看酒店详细信息

### 浏览建议
1. 从 **登录页** 开始
2. 登录后进入 **首页**，了解系统概览
3. 查看 **爬取任务** 相关页面，了解任务管理流程
4. 查看 **数据查询** 和 **数据详情**，了解数据管理功能
