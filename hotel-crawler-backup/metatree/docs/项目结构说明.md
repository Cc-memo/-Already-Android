# 项目结构说明

## 目录结构

```
metatree/
├── README.md                    # 项目主说明文档
├── docs/                        # 文档目录
│   ├── 需求文档.md              # 原始需求文档
│   ├── 需求文档_完善版.md       # 完善后的需求文档
│   ├── 产品规划.md              # 产品规划文档
│   └── 项目结构说明.md          # 本文件
└── prototypes/                  # HTML原型文件目录
    ├── login.html               # 登录页面
    ├── dashboard.html           # 首页/仪表盘
    ├── crawl-tasks.html         # 爬取任务列表页
    ├── create_task.html         # 创建爬取任务页
    ├── task_detail.html         # 任务详情页
    ├── data_query.html          # 数据查询页
    ├── data_detail.html         # 数据详情页
    └── task_list.html           # 任务列表页（备用）
```

## 文件说明

### 文档文件（docs/）

#### 需求文档.md
- **用途**：原始需求文档
- **内容**：项目背景、产品目标、功能需求、非功能需求等
- **适用对象**：产品经理、开发人员

#### 需求文档_完善版.md
- **用途**：完善后的需求文档
- **内容**：
  - 用户角色分析（普通用户、操作员、管理员）
  - 功能需求详细设计
  - 页面结构设计
  - UI/UX设计规范
  - 数据模型设计
  - API接口设计概要
- **适用对象**：产品经理、UI/UX设计师、开发人员

#### 产品规划.md
- **用途**：产品规划文档
- **内容**：
  - 功能架构
  - 页面结构规划
  - 交互流程设计
  - UI/UX设计规范
  - 页面详细设计
- **适用对象**：产品经理、UI/UX设计师

### 原型文件（prototypes/）

所有HTML原型文件均为独立的静态页面，可以直接在浏览器中打开查看。

#### login.html
- **页面名称**：登录页
- **功能**：用户登录入口
- **关键元素**：
  - 用户名/密码输入
  - 记住我选项
  - 忘记密码链接

#### dashboard.html
- **页面名称**：首页/仪表盘
- **功能**：系统概览和快速操作
- **关键元素**：
  - 数据统计卡片（总任务数、成功任务、总数据量、今日新增）
  - 快速爬取表单（支持多酒店批量输入、多平台选择）
  - 最近任务列表
  - 数据趋势图表

#### crawl-tasks.html
- **页面名称**：爬取任务列表页
- **功能**：查看和管理所有爬取任务
- **关键元素**：
  - 任务筛选（状态、平台、时间范围）
  - 任务列表表格
  - 任务操作（查看详情、暂停、继续、取消、重试）
  - 分页组件

#### create_task.html
- **页面名称**：创建爬取任务页
- **功能**：创建新的爬取任务（手动/定时）
- **关键元素**：
  - 任务类型选择（立即执行/定时执行）
  - 酒店名称输入（支持批量）
  - 平台选择（多选）
  - 爬取类型选择
  - 定时设置（如选择定时任务）

#### task_detail.html
- **页面名称**：任务详情页
- **功能**：查看任务详细信息和执行日志
- **关键元素**：
  - 任务基本信息
  - 任务执行状态
  - 爬取进度
  - 执行日志
  - 错误信息（如有）

#### data_query.html
- **页面名称**：数据查询页
- **功能**：查询和筛选酒店数据
- **关键元素**：
  - 多条件查询表单
  - 查询结果列表/卡片
  - 排序和筛选
  - 批量操作

#### data_detail.html
- **页面名称**：数据详情页
- **功能**：查看酒店详细信息
- **关键元素**：
  - 酒店基本信息
  - 多平台数据对比（标签页）
  - 价格趋势图表
  - 点评趋势图表
  - 房型信息列表

#### task_list.html
- **页面名称**：任务列表页（备用）
- **说明**：与crawl-tasks.html功能类似，可作为备用版本

## 设计规范

### 色彩系统
- **主色**：#1890FF（蓝色）
- **成功色**：#52C41A（绿色）
- **警告色**：#FAAD14（橙色）
- **错误色**：#F5222D（红色）
- **文字色**：#262626（主）、#595959（次）、#8C8C8C（辅助）
- **背景色**：#F0F2F5（主背景）、#FFFFFF（卡片背景）

### 布局规范
- **侧边栏宽度**：200px
- **内容区**：自适应宽度
- **卡片圆角**：8px
- **按钮圆角**：4px
- **基础间距单位**：8px

### 组件规范
- **按钮**：主要按钮（蓝色背景）、次要按钮（白色背景+边框）
- **输入框**：圆角4px，边框1px
- **表格**：斑马纹，悬停高亮
- **卡片**：白色背景，阴影效果

## 使用指南

### 查看原型
1. 直接在浏览器中打开 `prototypes/` 目录下的HTML文件
2. 建议从 `login.html` 开始浏览
3. 所有页面均为静态原型，包含基本的交互效果

### 开发参考
1. 参考 `docs/需求文档_完善版.md` 了解详细功能需求
2. 参考 `docs/产品规划.md` 了解页面结构和交互流程
3. 参考原型文件了解UI设计和交互效果

### 后续开发
1. 前端开发：建议使用 Vue.js + Element UI 或 React + Ant Design
2. 后端开发：建议使用 Spring Boot 或 Django
3. 爬虫开发：建议使用 Python Scrapy + Selenium
