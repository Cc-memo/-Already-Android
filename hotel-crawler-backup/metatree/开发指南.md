# 开发指南

## 环境设置

### 1. 创建虚拟环境

```bash
# 方式1: 使用脚本（推荐）
./setup_dev.sh

# 方式2: 手动创建
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# 或
venv\Scripts\activate  # Windows
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置环境变量

复制 `.env.example` 为 `.env` 并修改配置：

```bash
cp .env.example .env
```

编辑 `.env` 文件，设置数据库等配置。

## 开发工作流

### 激活虚拟环境

```bash
# 方式1: 使用脚本
./run_dev.sh

# 方式2: 手动激活
source venv/bin/activate
```

### 运行爬虫

```bash
# 使用主程序
python crawler/main.py --hotel "北京饭店" --platforms meituan ctrip

# 或使用开发脚本
./run_dev.sh python crawler/main.py --hotel "北京饭店"
```

### 运行测试

```bash
# 运行所有测试
python -m pytest tests/ -v

# 运行特定测试
python -m pytest tests/test_meituan_spider.py -v

# 或使用开发脚本
./run_dev.sh python -m pytest tests/
```

### 开发单个平台爬虫

```python
from crawler.spiders.meituan_spider import MeituanSpider

# 使用上下文管理器（推荐）
with MeituanSpider(username='user', password='pass') as spider:
    # 登录
    spider.login()
    
    # 搜索酒店
    hotels = spider.search_hotel("北京饭店")
    print(f"找到 {len(hotels)} 个酒店")
    
    # 获取详情
    for hotel in hotels[:1]:  # 只获取第一个
        if hotel.get('hotel_url'):
            detail = spider.get_hotel_detail(hotel['hotel_url'])
            print(detail)
```

## 开发新平台爬虫

1. 在 `crawler/spiders/` 创建新文件，例如 `new_platform_spider.py`
2. 参考现有爬虫的结构实现：
   - `__init__()` - 初始化
   - `login()` - 登录方法
   - `search_hotel()` - 搜索酒店
   - `get_hotel_detail()` - 获取详情
   - `crawl_hotel()` - 完整流程
3. 在 `crawler/main.py` 中注册新平台
4. 创建对应的测试文件

## 调试技巧

### 1. 使用非无头模式

在爬虫初始化时设置 `headless=False`：

```python
self.browser = BrowserManager(headless=False)
```

### 2. 添加延迟

在关键操作后添加延迟，观察浏览器行为：

```python
time.sleep(3)  # 暂停3秒
```

### 3. 保存页面截图

```python
self.browser.driver.save_screenshot('debug.png')
```

### 4. 打印页面源码

```python
print(self.browser.get_page_source())
```

## 常见问题

### 1. Chrome驱动问题

确保已安装Chrome浏览器，ChromeDriver会自动下载或手动指定路径。

### 2. 选择器失效

各平台页面可能更新，需要及时更新CSS选择器。

### 3. 反爬机制

- 增加请求间隔
- 使用代理IP
- 轮换User-Agent
- 模拟真实用户行为
