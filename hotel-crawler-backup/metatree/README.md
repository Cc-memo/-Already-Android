# 酒店信息爬取及管理平台

## 项目简介

酒店信息爬取及管理平台是一个高效、精准、智能的酒店数据管理解决方案。系统支持对美团、携程、飞猪、高德四大主流平台的酒店信息进行自动化爬取，并提供完善的数据管理、分析、导出功能。

## 功能特性

### 核心功能模块

1. **用户认证模块**
   - 用户登录/登出
   - 密码修改
   - 权限管理（普通用户、操作员、管理员）

2. **数据爬取模块**
   - 手动爬取任务创建
   - 定时爬取任务设置
   - 批量酒店爬取（支持多酒店、多平台）
   - 任务状态监控（等待中、爬取中、已完成、已失败、已暂停）
   - 任务进度显示
   - 任务操作（暂停、继续、取消、重试）

3. **数据管理模块**
   - 多条件组合查询
   - 数据列表/卡片/详情展示
   - 数据排序和筛选
   - 数据修正功能

4. **数据分析模块**
   - 价格分析（平台对比、趋势分析）
   - 点评分析（分数趋势、关键词提取、情感分析）
   - 预订量分析（平台对比、趋势分析、高峰时段）

5. **数据导出模块**
   - 多格式导出（Excel、CSV）
   - 导出历史记录
   - 批量导出

6. **用户管理模块**（仅管理员）
   - 用户创建/编辑/删除
   - 角色权限分配
   - 用户状态管理

7. **系统设置模块**（仅管理员）
   - 爬取参数设置
   - 存储设置
   - 日志管理

## 用户角色

### 普通用户（User）
- 数据查询与查看
- 数据导出
- 个人设置

### 操作员（Operator）
- 普通用户所有权限
- 创建爬取任务
- 管理爬取任务
- 查看爬取日志

### 管理员（Admin）
- 操作员所有权限
- 用户管理
- 系统设置
- 数据备份与清理

## 开发说明

### 原型特点

1. **完整的页面设计**：包含所有核心功能页面
2. **统一的视觉风格**：采用一致的设计规范和色彩系统
3. **交互效果**：包含基本的JavaScript交互逻辑
4. **响应式布局**：适配不同屏幕尺寸

### 后续开发建议

1. **前端框架**：建议使用 Vue.js 或 React 进行实际开发
2. **UI组件库**：可使用 Element UI 或 Ant Design
3. **后端开发**：建议使用 Spring Boot 或 Django
4. **数据库**：MySQL（结构化数据）+ MongoDB（非结构化数据）
5. **爬虫框架**：Python Scrapy + Selenium
